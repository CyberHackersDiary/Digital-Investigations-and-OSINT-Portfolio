# Disk Forensics
Examination of storage media — hard drives, SSDs, USB devices, and disk images — to recover evidence of user activity, system events, and malicious behaviour. Includes recovery of deleted or concealed data with strict adherence to evidence integrity.

## Skills Demonstrated
* Disk Image Acquisition & Verification (hashing, write-blocking)
* File System Analysis (NTFS, FAT32, exFAT, ext4, APFS, HFS+)
* Partition & Volume Structure Examination
* Deleted File Recovery & Carving
* MFT & Inode Analysis
* Metadata & Timestamp Analysis (MACB times)
* Registry Analysis & Artefact Extraction
* Prefetch, LNK & Shellbag Analysis
* Browser History & Web Artefact Recovery
* Email & Communication Artefact Analysis
* Windows Event Log & Syslog Analysis
* Steganography & Hidden Data Detection
* Encryption & BitLocker Artefact Analysis
* Super-Timeline Creation & Reconstruction
* Anti-Forensics Detection (wiping, timestomping, obfuscation)
* Evidence Preservation & Chain of Custody
