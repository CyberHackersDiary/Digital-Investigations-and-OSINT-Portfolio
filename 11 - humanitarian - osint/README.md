# Humanitarian OSINT

Applying open-source intelligence techniques to support humanitarian efforts, crisis response, disaster analysis, and the verification of publicly available information during emergencies.

This repository documents my journey in Humanitarian OSINT through practical case studies, investigative methodologies, and real-world analysis.

## Areas of Focus

* Crisis Mapping
* Disaster Response Analysis
* Event Verification
* Conflict Monitoring
* Humanitarian Research
* Situation Awareness
* Geospatial Intelligence
* Information Verification
* Open-Source Data Analysis
* Evidence Documentation & Reporting

## Investigation Methodology

1. Identify the humanitarian event or crisis.
2. Collect publicly available information from multiple sources.
3. Verify the accuracy and credibility of information.
4. Analyze geospatial, visual, and contextual evidence.
5. Correlate findings across independent sources.
6. Document conclusions with supporting evidence.
