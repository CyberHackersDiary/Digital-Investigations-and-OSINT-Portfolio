#OSINT & Digital Investigations Portfolio

Open-source intelligence investigations spanning geolocation, media verification, social networks, threat intelligence, corporate research, fraud, and humanitarian analysis — each case documented to professional reporting standards.

Every investigation in this portfolio is independently conducted from scratch. Evidence is collected, verified, corroborated, and synthesized into structured intelligence assessments using reproducible, professional-grade methodologies.

Investigative Methodology
Every case follows a structured pipeline designed for rigor and reproducibility:
Research Question → Collection → Verification → Correlation
       → Analysis → Findings → Intelligence Assessment → Report
Each stage emphasizes source validation, evidence corroboration, and documented reasoning — the same standards applied in professional investigative and intelligence contexts.
