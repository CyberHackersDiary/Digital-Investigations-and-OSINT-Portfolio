# Digital Forensics & Incident Response (DFIR) Portfolio
Hands-on DFIR investigations spanning network forensics; memory analysis; disk examination; threat hunting; and malware analysis. Each case is independently worked through and documented from scratch — evidence collected, analysed, and mapped to real-world attack frameworks.

## Area of Focus
| Domain                 | Description                                                                      |
| ---------------------- | -------------------------------------------------------------------------------- |
| Network Forensics      | Packet-level investigation of malicious traffic; C2 detection; and communication reconstruction |
| Memory Forensics       | Live memory analysis to surface hidden processes; injected code; and volatile artefacts |
| Disk Forensics         | Deep file system examination; deleted file recovery; and forensic timeline analysis |
| Threat Hunting         | IOC-driven; hypothesis-based hunting to detect threats that evade automated detection |
| Malware Analysis       | Binary dissection using static and dynamic techniques to profile attacker tooling and behaviour |
