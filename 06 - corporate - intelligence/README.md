# Domain Investigations

Analyzing domains, DNS records, and internet infrastructure using open-source intelligence and evidence-based investigation techniques.

This repository documents my journey in domain investigations through practical case studies, investigative methodologies, and real-world analysis.

## Areas of Focus

* Domain Profiling
* DNS Analysis
* WHOIS Research
* Infrastructure Mapping
* Subdomain Enumeration
* Historical Analysis
* Reputation Assessment
* Threat Detection
* Attribution Analysis
* Evidence Documentation & Reporting

## Investigation Methodology

1. Identify the target domain.
2. Collect domain and DNS information.
3. Analyze infrastructure and ownership indicators.
4. Correlate findings across multiple sources.
5. Assess legitimacy and risk.
6. Document conclusions with supporting evidence.
