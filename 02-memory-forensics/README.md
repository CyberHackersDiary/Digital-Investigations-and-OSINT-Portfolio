# Memory Forensics
Acquisition and analysis of volatile memory (RAM) to uncover malicious activity invisible on disk. Reveals running processes, injected code, decrypted payloads, network state, and credentials — critical for detecting fileless malware and rootkits.

## Skills Demonstrated
* Memory Acquisition & Imaging
* Full Dump, Hibernation File & Crash Dump Analysis
* Process & Thread Analysis
* DLL & Module Inspection
* Code Injection Detection (hollowing, DLL injection, reflective loading)
* Rootkit & Stealth Technique Detection
* Fileless Malware Identification
* Kernel & Driver Analysis
* Network Connection Artefact Extraction
* Registry Hive Analysis from Memory
* Credential & Secret Extraction
* Malware Unpacking & Payload Recovery
* String & Pattern Extraction
* Volatile Artefact Timeline Reconstruction
* Volatility Framework & Plugin Usage
* IOC Extraction & Correlation
